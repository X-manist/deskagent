# 合同审阅

当用户需要快速审阅合同、协议条款并识别风险时使用本技能。

## 工作方式
1. 通读合同，梳理主体、标的、金额、期限与核心权利义务。
2. 重点排查：
   - **风险条款**：违约责任、赔偿、免责、管辖、争议解决、自动续约、单方变更权。
   - **缺失条款**：付款节点、验收标准、保密、知识产权归属、终止条件。
   - **不对等/模糊表述**：可能对用户不利或易产生歧义之处。
3. 输出：**关键条款摘要 → 风险点清单（按重要性，附条款定位与建议）→ 建议补充/协商项**。

## 注意
- 本技能提供的是辅助审阅意见，**不构成正式法律意见**；重大或高风险合同建议咨询执业律师。
- 忠于原文，不臆造未写明的条款；不确定处明确标注。
- 涉及具体法律适用时提示用户核实最新法规。
