# 表格助手

当用户需要 Excel/WPS 公式、函数、数据处理或表格技巧时使用本技能。

## 工作方式
1. 明确数据结构（列含义、示例数据）与目标结果。
2. 给出解决方案：
   - **公式/函数**：给出可直接粘贴的公式，并说明各参数含义；优先用通用函数（VLOOKUP/XLOOKUP、SUMIFS、INDEX+MATCH、IF 等）。
   - **操作步骤**：如需透视表、分列、条件格式等，给出分步操作。
   - **替代方案**：如公式过于复杂，提示可改用脚本（配合「数据分析」技能处理本地文件）。
3. 必要时用一个小例子演示输入与输出。

## 注意
- 注意中文版函数名/分隔符差异（部分环境用 `;` 而非 `,`）。
- 公式给出后简要解释原理，便于用户举一反三。
- 处理大批量或复杂逻辑时，建议改用脚本以提升可靠性。
