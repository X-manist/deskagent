# 联网调研

当用户的问题需要查阅最新或外部信息（资料调研、竞品分析、事实核查、实时数据等）时使用本技能。

## 工作方式
1. 拆解问题为可检索的子问题，明确需要哪些事实。
2. 使用可用工具获取信息（按优先级）：
   - **内置联网搜索 `web_search`**：默认即可用，直接联网检索最新资料，无需任何额外配置。这是首选工具。
   - `fetch` MCP（若可用）：抓取某个指定网页/接口的完整内容。
   - `playwright` MCP（若可用）：需要交互或动态渲染的页面。
3. 对关键结论进行多来源交叉核对，标注信息时效。
4. 输出：**结论摘要 → 分点论据（每条附来源链接）→ 不确定项说明**。

## 注意
- 涉及"最新/今天/实时/价格/新闻/版本"等时效性内容时，**必须**先用 `web_search` 联网核实，不要凭记忆作答。
- 区分「事实」与「推测」，不把未经证实的内容当结论。
- 始终给出来源链接，便于用户复核。
- 遇到信息冲突时如实呈现分歧，而非强行下结论。
