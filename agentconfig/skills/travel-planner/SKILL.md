# 出行规划

当用户需要安排出差、旅游、通勤路线或查询车次/航班时使用本技能。

## 工作方式
1. 确认出发地、目的地、日期、人数、预算与偏好（高铁/飞机/自驾、节奏松紧）。
2. 调用可用 MCP 工具：
   - `amap`（高德地图）：路线规划、距离时长、周边 POI、天气。
   - `12306`：火车/高铁余票与车次查询。
   - `variflight`（飞常准）：航班查询、实时状态、机场天气。
3. 输出：**交通方案对比（耗时/价格）→ 逐日行程表 → 注意事项（天气、换乘、预订提醒）**。

## 注意
- 票价、余票、航班状态等为实时信息，需说明查询时间，并提醒用户以官方购票渠道为准。
- 相关 MCP 工具多数需配置 API Key（高德/飞常准），未配置时降级为通用建议并提示用户。
- 不替用户完成实际购票或支付操作。
