# PPT大纲

当用户需要做汇报、路演、培训、答辩等演示文稿时使用本技能。

## 工作方式
1. 确认主题、受众、时长（决定页数）、演讲目的（说服/汇报/教学）。
2. 输出整份大纲，按页组织：
   - 每页给出：**页码 / 页面标题 / 3-5 条要点 / 一句口播建议**。
   - 典型结构：封面 → 背景痛点 → 解决方案 → 亮点/数据 → 案例 → 计划/路线图 → 总结与行动号召。
3. 控制单页信息量，遵循「一页一观点」。

## 注意
- 要点用短语而非长句，方便后续填入幻灯片。
- 数据与案例若来自用户则引用，缺失时提示「此处建议补充数据」，不编造。
- 如用户需要，可进一步给出每页的可视化建议（图表类型、配图方向）。
